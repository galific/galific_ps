Requirements

Inveo PrestaShop Module has a few requirements which must be met before you are able to install and use it.

  • properly installed and set up PrestaShop 1.3.x.x/1.4.x.x/1.5.x.x/1.6.x.x
  • PHP 5 and later (Safe Mode is supported)

New installation

  1. Decompress the PrestaShop Module archive to a local directory on your system.

  2. Upload all the files contained in this archive (retaining the directory structure and excluding the docs folder) to your server or hosting account with BINARY mode for transfer (do not use ASCII/Text mode or critical files get corrupted).

  3. Please go to PrestaShop admin panel - Modules tab and click the Install button.

  4. The installation process is finished.

Upgrade

  1. Decompress the PrestaShop Module archive to a local directory on your system.

  2. Upload all the files contained in this archive (retaining the directory structure and excluding the docs folder) to your server or hosting account with BINARY mode for transfer (do not use ASCII/Text mode or critical files get corrupted) adn overwrite all exiting files.

  3. Please go to PrestaShop admin panel - Modules tab and click the Install button.

  4. The upgrade process is finished.
